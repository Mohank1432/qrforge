# 🔲 QRForge — Free QR Code Generator

A beautiful, free QR code generator with customization options. Built with vanilla JavaScript and Vite.

![QRForge Screenshot](https://img.shields.io/badge/Status-Live-brightgreen)

## ✨ Features

- **4 Content Types** — URL, Text, WiFi credentials, Email
- **Full Customization** — Colors, size (128–1024px), error correction, margin
- **Download** — PNG, SVG, or copy to clipboard
- **Bulk Generation** — Generate up to 50 QR codes at once
- **History** — Recent QR codes saved locally
- **Premium UI** — Dark glassmorphism theme with smooth animations
- **SEO Optimized** — Meta tags, JSON-LD, Open Graph
- **Ad Ready** — Pre-placed zones for Google AdSense

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 📦 Build for Production

```bash
npm run build
npm run preview
```

## 🛠️ Tech Stack

- **Vite** — Build tool
- **Vanilla JS** — No framework bloat
- **QRCode.js** — QR code generation
- **CSS** — Custom design system with glassmorphism
